相关概念介绍

Ruby

Ruby本身的特点就是简洁、易懂。Ruby可以用来做Web开发，Ruby on Rails就是用Ruby写的一个开源Web框架。

Jekyll

  • Jekyll是一款静态页面生成工具。
  • Octopress就是使用的Jekyll生成博客网页。
  • GitHub Pages、GitCafe Pages都支持Jekyll服务。

Rake

Rake-Ruby Make可以嵌入Ruby代码的命令行任务管理工具。像rake new_post、rake generake等命令行任务都是在Rakefile中定义的。
